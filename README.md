# 🎬 AI 做视频 · 从零到一条片子

一份面向零基础的交互式教程网页,带你把「用 AI 做视频」的完整链路走一遍。

**在线预览:** https://youmin7003pro-hash.github.io/ai_video/

## 内容包含

- **核心心法** —— 「AI 不是导演,是特效外包」:把片子拆成 5–10 秒短镜头逐一生成
- **两条主路线** —— 文生视频 vs 图生视频,以及为什么新手该从图生视频入手
- **工具地图** —— 按用途分四类(生成画面 / 分镜图 / 配音配乐 / 剪辑数字人),并标注「国内直连 / 需梯子」
- **六步流水线** —— 脚本 → 关键帧 → 图生视频 → 配音 → 剪辑 → 检查导出
- **提示词生成器** —— 可交互组件,点选「主体/动作/镜头/光线/风格」实时拼出提示词,一键复制
- **镜头运动术语速查** —— 推 / 拉 / 摇 / 环绕 / 航拍…… 中英对照
- **上手项目** —— 「15 秒咖啡短片」五步实操清单(可点击打勾,带完成进度)
- **避坑清单** —— 新手最常踩的六个坑及解法

## 技术

单个 `index.html`,零依赖、零构建。纯 HTML + CSS + 原生 JS,内含胶片颗粒、深浅色主题切换与滚动动画。直接双击即可打开,或用 GitHub Pages 托管。

## 设计

电影剪辑室美学 —— 经典橙青(teal & orange)电影调色 + 胶片颗粒 + 时间码/场记板语言,把「做视频」呈现为一条拍摄流水线。
